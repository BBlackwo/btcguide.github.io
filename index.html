--- title: 10x Security Bitcoin Guide ---

How to store bitcoin without any single point of failure.

Purpose

Multisig security is a difference in kind and not in degree. The ability to avoid loss while making 1 (or more) catastrophic failures in securing your bitcoin is unrivaled by any other security offering. By using a system that is fundamentally much more secure, you can move much faster (with less caution) through each step and still attain far higher levels of security vs any single-key system. This guide will show you how.

The ultimate purpose of this guide is to make multisig accessible to a tech-savvy audience who does not write code. Unfortunately, right now there is 1 step that requires using the command-line. For users who are not comfortable using the command-line, you are encouraged to wait for a future release.
